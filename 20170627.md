Using strings in lists in functions

```py
n = ["Michael", "Lieberman"]
def join_strings(words):
    result = ""
    for i in words:
        result = result + i
    return result
print join_strings(n)
```

Using a list of lists in a function

```python
n = [[1, 2], [4, 5, 6, 7]]
def flatten(lists):
    results = [ ]
    for x in lists:
        for j in x:
            results.append(j)
    return results

print flatten(n)
```
